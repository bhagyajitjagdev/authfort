## Server

Requires Python 3.11+ and a SQL database (PostgreSQL, SQLite, or MySQL).

### PostgreSQL (recommended)

<PackageInstall syncKey="py" variants={[
  { label: "pip", command: "pip install authfort" },
  { label: "uv", command: "uv add authfort" },
]} />

### SQLite

<PackageInstall syncKey="py" variants={[
  { label: "pip", command: "pip install authfort[sqlite]" },
  { label: "uv", command: "uv add authfort[sqlite]" },
]} />

### MySQL

<PackageInstall syncKey="py" variants={[
  { label: "pip", command: "pip install authfort[mysql]" },
  { label: "uv", command: "uv add authfort[mysql]" },
]} />

The base install includes `asyncpg` for PostgreSQL. The extras install `aiosqlite` or `aiomysql` respectively.

## Service SDK

For microservices that need to verify JWTs. No database required.

<PackageInstall syncKey="py" variants={[
