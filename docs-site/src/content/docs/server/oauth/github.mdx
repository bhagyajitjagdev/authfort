export const configExample = `from authfort import GitHubProvider

GitHubProvider(
    client_id="your-github-client-id",
    client_secret="your-github-client-secret",
    scopes=("read:user", "user:email"),  # default
)`;

## Setup

<Steps>

1. Go to [GitHub Developer Settings](https://github.com/settings/developers) → OAuth Apps → New
2. Set the Authorization callback URL: `https://yourapp.com/auth/oauth/github/callback`
3. Copy the Client ID and Client Secret

</Steps>

## Configuration

<CodeBlock code={configExample} />

### Scopes

| Scope | Description |
|-------|-------------|
| `read:user` | Access the user's profile (name, avatar) |
| `user:email` | Access the user's email address |

The default scopes cover everything AuthFort needs. Only customize if you need additional GitHub API access.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/auth/oauth/github/authorize` | Start GitHub OAuth flow |
| GET | `/auth/oauth/github/callback` | Handle GitHub callback |
